## Class vs Struct 





| 名称     | class                                                        | struct                    |
| -------- | ------------------------------------------------------------ | ------------------------- |
|          |                                                              |                           |
| 继承     | 可以继承，可以被final修饰                                    | 不能继承，不能被final修饰 |
| 内存     | 自动引用计数                                                 | stack栈，自动管理         |
| 传值     | 指针传递                                                     | 值传递 copy on write      |
| 函数派发 | 1 初始声明 函数表派发 2 extensions 直接派发 3 extensions with @objc 消息派发 | 直接派发                  |
