## PrettyJSON



插件安装后，可以通过 command + shift + p ，搜索插件进行调用



修改快捷键。

Sublime -> Perferences -> Key Bindings ,在右侧的空白添加自己的快捷键就可以了。 



```
[
	{
    	"keys": [
      	  "ctrl+command+j"
    	], 
    	"command": "pretty_json"
	}
]
```



修改成功后， command + shift + p 找到对应的插件，会显示对应的快捷键。
